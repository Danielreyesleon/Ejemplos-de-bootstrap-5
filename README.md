# Ejemplos de Bootstrap 5

Este repositorio reúne ejemplos **básicos** usados en una demostración de Bootstrap 5, enfocados en cómo instalarlo y usar sus componentes principales.

## Instalación rápida

### Opción 1: CDN (la más simple)
Agrega esto dentro de tu `<head>`:

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
```

Y antes de cerrar `</body>`:

```html
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

### Opción 2: npm

```bash
npm install bootstrap
```

## Uso básico

Ejemplo mínimo de botón con estilos de Bootstrap:

```html
<button class="btn btn-primary">Hola Bootstrap 5</button>
```
